# Kotlin Null Safety Exercises

This repository contains two beginner-friendly Kotlin exercises focused on null safety, user input validation, and error handling.

## Projects

### 1. SafeEmailSender

A simple program that accepts an email address from the user and checks whether the input is valid (not empty or blank) before simulating the sending of an email.

#### Features
- Accepts user input
- Checks for empty or blank values
- Demonstrates Kotlin null safety
- Displays success or error messages

---

### 2. ParseNumber

A program that safely converts user input into an integer using Kotlin's `toIntOrNull()` function.

#### Features
- Accepts user input
- Safely parses numbers
- Handles invalid input without crashing
- Demonstrates nullable types and null checking

---
